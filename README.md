For insurance providers to analyze and reduce employee attrition.

**Files (keep at repo root):**
- `app.py`

- `requirements.txt` *(no versions pinned)*
- `README.md`

**Deploy on Streamlit Cloud**
1. Create a GitHub repo and upload these files.
2. In Streamlit Cloud, set **Main file** to `app.py`.
3. Upload your CSV (must include `Attrition`/`attribution`).

**Local run**
```bash
pip install -r requirements.txt
streamlit run app.py
```
